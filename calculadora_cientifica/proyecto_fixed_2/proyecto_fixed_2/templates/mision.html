MISION

Desarrollar una calculadora científica web que brinde a los usuarios una herramienta confiable, rápida y fácil de usar para realizar cálculos matemáticos básicos y avanzados. Mediante el uso de tecnologías web como HTML, CSS y Python con Flask, se busca ofrecer una plataforma accesible desde cualquier dispositivo, que permita resolver operaciones con precisión, apoyar el aprendizaje de las matemáticas y mejorar la productividad de estudiantes, docentes y profesionales.
