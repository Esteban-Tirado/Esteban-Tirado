### 📈 Mis Proyectos Destacados
### 📐 Calculadora Científica
Aplicación web interactiva para resolver operaciones matemáticas complejas directamente en el navegador.

Una interfaz limpia y lógica construida con JavaScript puro. No fue solo un ejercicio de lógica; me enfoqué en que la experiencia de usuario fuera fluida, manejando correctamente los estados de las operaciones y la precedencia matemática de los operadores.

Incluye un panel completo de funciones científicas (trigonometría, logaritmos, potencias), un historial de operaciones recientes, diseño responsivo adaptado a móviles y computadoras, y un sistema de validación de errores en tiempo real para evitar fallos al presionar teclas consecutivas.

Lo que más aprendí: estructurar la lógica compleja de JavaScript sin depender de librerías externas, manipular el DOM de forma avanzada para mantener la pantalla actualizada y controlar eventos de teclado para que responda como una calculadora física real.


### 🔄 Tarjeta con Movimiento 360°
Experimento visual interactivo y moderno que desafía los límites del diseño web con CSS avanzado.

Un componente interactivo diseñado para capturar la atención del usuario. El objetivo principal de este proyecto fue dominar los efectos visuales y las perspectivas en tres dimensiones aplicadas al desarrollo de interfaces (UI/UX) de alta calidad.

Incluye efectos de inclinación tridimensional (efecto *tilt*) que reaccionan al movimiento del cursor del mouse, transiciones suaves de color y sombras dinámicas que simulan el reflejo de la luz, y animaciones optimizadas para mantener un rendimiento alto en el navegador.

Lo que más aprendí: dominar las propiedades de transformación 3D de CSS (`transform`, `perspective`), coordinar eventos de JavaScript con estilos en tiempo real para rastrear la posición del mouse, y diseñar interfaces inmersivas con un aspecto premium y moderno.
